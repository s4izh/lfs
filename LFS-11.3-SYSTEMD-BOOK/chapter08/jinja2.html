8.71. Jinja2-3.1.2

Jinja2 is a Python module that implements a simple pythonic template language.

Approximate build time: less than 0.1 SBU
Required disk space: 3.4 MB

8.71.1. Installation of Jinja2

Build the package: 

pip3 wheel -w dist --no-build-isolation --no-deps $PWD

Install the package: 

pip3 install --no-index --no-user --find-links dist Jinja2

8.71.2. Contents of Jinja2

Installed directory: /usr/lib/python3.11/site-packages/Jinja2-3.1.2.dist-info
