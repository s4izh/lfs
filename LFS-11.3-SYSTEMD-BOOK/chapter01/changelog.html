1.3. Changelog

This is version 11.3-systemd of the Linux From Scratch book, dated March 1st, 2023. If this book is more than six months old, a newer and better version is probably already available. To find out, please check one of the mirrors via https://www.linuxfromscratch.org/mirrors.html.
